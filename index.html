CareBook Online Website Merupakan website yang saya buat untuk memenuhi penilaian UAS mata kuliah Human Computer Interaction pada semester 2. Saya diberikan tugas untuk membuat sebuah tampilan website yang dapat digunakan untuk melakukan booking kunjungan ke Rumah Sakit secara online.

DirectNews.com

DirectNews.com Merupakan final project Human Computer Interaction yang saya buat di semester 2. Website ini merupakan website yang dapat menampilkan berbagai macam berita dilengkapi dengan berbagai macam media seperti : Lagu, Gambar dan Video.

NewsID Mobile

Merupakan Aplikasi berbasis Android yang dibangun menggunakan bahasa java. Aplikasi ini berfungsi sebagai platform pembaca berita dengan berbagai kategori yang tersedia. Aplikasi ini merupakan final project yang saya dan teman sekelompok saya kerjakan pada semester 4 dalam matakuliah Mobile Object Oriented Programming. Untuk mendowload aplikasi NewsID silahkan klik disini. Tonton video presentasi dari NewsId.

SanitizeYou

Merupakan sebuah produk embed menggunakan arduino uno. Sanitizeyou merupakan smart sanitizer yang dapat melakukan pengecekan suhu juga mengeluarkan cairan sanitize secara otomatis apabila pada saat pengecekan suhu user tidak melebihi batas maksimum. Produk ini dirancang dalam pemenuhan final project pada mata kuliah Embedded System dan Software Engineering pada semester 4. Care Book Mobile

Merupakan rancangan/prototype dari aplikasi booking kunjungan rumah sakit berbasis mobile. Melalui aplikasi ini user bisa memeriksa ketersediaan jadawal kunjungan, dan user akan menerima sebuah barcode sebagai tanda bahwa user sudah terdaftar dalam list kunjungan. Aplikasi ini saya buat dalam memenuhi tugas ujian pada semester 2.

Merupakan sebuah design UI/UX yang saya rancang bersama teman sekelompok saya untuk mengikuti lomba Integer Undika "Smart City Smart People" Aplikasi Hunting dirancang semacam aplikasi tour guide yang juga dapat menjadi wadah bagi pengusaha-pengusaha tour guide.
